# Квест «Куда утекают твои деньги»

Интерактивный одностраничный сайт-квест на чистом **HTML**, **CSS** и **JavaScript** (без сборки и зависимостей).

## Как открыть

Откройте в браузере файл `index.html` (двойной клик или «Открыть с помощью» → браузер).

Либо из папки проекта поднимите локальный сервер, если удобнее:

```bash
npx serve .
```

## Структура

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Разметка: карточка сцены, промежуточный экран, подпись внизу |
| `styles.css` | Внешний вид (тёмная палитра, золото / electric-blue, кнопки) |
| `script.js` | Данные сцен и логика переходов |
| `images/` | Иллюстрации сцен и финал (`scene4(a).jpeg` и др.; пути в `script.js`) |

## Что править

- **Тексты и кнопки** — массив `scenes` в `script.js`.
- **Ссылка на форму** после квеста — у последней сцены с полем `actionUrl` (сейчас `#`); замените на `https://…` вашей формы.
- **Картинки** — замените файлы в `images/` и пути `image: "images/…"` в каждой сцене.

## Логика

- Стартовый экран с кнопкой «Найти мои деньги» ведёт на первую сцену без промежуточного экрана «Следствие установило».
- На остальных сценах после выбора кнопки: плавное исчезновение → экран «Следствие установило: …» на несколько секунд → следующая сцена.
- Финальная кнопка открывает новую вкладку (указанный `actionUrl`).
